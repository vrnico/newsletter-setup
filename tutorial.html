Step-by-Step Tutorial

Set up your free Gmail newsletter in about 20 minutes

Part 1: Set Up Your Google Sheet

First, we'll create a spreadsheet to store subscriber emails

1.1

Create a New Google Sheet

  1. Go to sheets.google.com
  2. Click the + Blank button to create a new spreadsheet
  3. Name it "My Newsletter Subscribers" (click on "Untitled spreadsheet" at the top)

Your Google Sheet should look like this:

A B C D
email name dateJoined status
1.2

Add Column Headers

In row 1, add these headers (one per column):

Column A: email
Column B: name
Column C: dateJoined
Column D: status

Why these columns?

email - Where we send the newsletter
name - To personalize emails ("Hi Alex!")
dateJoined - Track when they subscribed
status - Active or Unsubscribed

1.3

Add Test Subscribers

Add 1-2 test emails (use your own email so you can test!):

email name dateJoined status
your.email@gmail.com Your Name 2024-01-15 Active

Part 2: Create the Apps Script

Now we'll write the code that sends emails

2.1

Open Apps Script

  1. In your Google Sheet, click Extensions in the menu bar
  2. Click Apps Script
  3. A new tab will open with the script editor
  4. Delete any code that's already there
2.2

Paste the Newsletter Code

Copy and paste this entire code into the script editor:

/**
 * GMAIL NEWSLETTER SCRIPT
 * This script sends emails to all active subscribers in your Google Sheet
 */

// Configuration - EDIT THESE!
const CONFIG = {
  senderName: "Your Name",           // Your name (appears in "From")
  replyTo: "your.email@gmail.com",   // Your email address
  sheetName: "Sheet1"                 // Name of the sheet tab
};

/**
 * Sends newsletter to all active subscribers
 */
function sendNewsletter() {
  // Get the spreadsheet and data
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(CONFIG.sheetName);
  const data = sheet.getDataRange().getValues();

  // Email content - EDIT THIS FOR EACH NEWSLETTER!
  const subject = "My Newsletter - Issue #1";
  const htmlBody = `
    <h1>Welcome to My Newsletter!</h1>
    <p>Hi {{NAME}},</p>
    <p>Thanks for subscribing! Here's what's new:</p>
    <ul>
      <li>Update #1: Something exciting</li>
      <li>Update #2: Another cool thing</li>
    </ul>
    <p>Best regards,<br>${CONFIG.senderName}</p>
    <hr>
    <small>You received this because you subscribed. Reply to unsubscribe.</small>
  `;

  let sentCount = 0;
  let errors = [];

  // Loop through each row (skip header row)
  for (let i = 1; i < data.length; i++) {
    const email = data[i][0];
    const name = data[i][1] || "there";
    const status = data[i][3];

    // Only send to active subscribers with valid emails
    if (status === "Active" && email && email.includes("@")) {
      try {
        // Personalize the email
        const personalizedBody = htmlBody.replace("{{NAME}}", name);

        // Send the email
        GmailApp.sendEmail(email, subject, "", {
          htmlBody: personalizedBody,
          name: CONFIG.senderName,
          replyTo: CONFIG.replyTo
        });

        sentCount++;
      } catch (e) {
        errors.push(email + ": " + e.message);
      }
    }
  }

  // Show results
  let message = `Newsletter sent to ${sentCount} subscribers!`;
  if (errors.length > 0) {
    message += `\n\nErrors:\n${errors.join("\n")}`;
  }
  SpreadsheetApp.getUi().alert(message);
}

/**
 * Sends a test email to yourself
 */
function sendTestEmail() {
  const testEmail = CONFIG.replyTo;
  const subject = "[TEST] My Newsletter";
  const htmlBody = `
    <h1>This is a test email!</h1>
    <p>Hi {{NAME}},</p>
    <p>If you can read this, your newsletter is working!</p>
    <p>Best regards,<br>${CONFIG.senderName}</p>
  `.replace("{{NAME}}", "Test User");

  GmailApp.sendEmail(testEmail, subject, "", {
    htmlBody: htmlBody,
    name: CONFIG.senderName
  });

  SpreadsheetApp.getUi().alert(`Test email sent to ${testEmail}!`);
}

/**
 * Adds a custom menu to the spreadsheet
 */
function onOpen() {
  SpreadsheetApp.getUi()
    .createMenu('Newsletter')
    .addItem('Send Test Email', 'sendTestEmail')
    .addItem('Send Newsletter to All', 'sendNewsletter')
    .addToUi();
}

/**
 * Handles form submissions from your website
 * This runs automatically when someone submits the form
 */
function doPost(e) {
  try {
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(CONFIG.sheetName);
    const data = JSON.parse(e.postData.contents);

    // Add new subscriber to sheet
    sheet.appendRow([
      data.email,
      data.name,
      new Date().toISOString().split('T')[0],
      "Active"
    ]);

    return ContentService.createTextOutput(JSON.stringify({
      success: true,
      message: "Subscribed successfully!"
    })).setMimeType(ContentService.MimeType.JSON);

  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({
      success: false,
      message: error.message
    })).setMimeType(ContentService.MimeType.JSON);
  }
}

// For GET requests (testing)
function doGet() {
  return ContentService.createTextOutput("Newsletter API is running!");
}
2.3

Update the Configuration

At the top of the code, find the CONFIG section and update it with your info:

const CONFIG = {
  senderName: "Your Actual Name",        // Change this!
  replyTo: "your.actual@gmail.com",     // Change this!
  sheetName: "Sheet1"                    // Change if your sheet tab has a different name
};
2.4

Save and Run

  1. Click the Save button (floppy disk icon) or press Ctrl+S
  2. Name the project "Newsletter Script" when prompted
  3. Click the function dropdown (says "sendNewsletter") and select onOpen
  4. Click Run
  5. You'll see an "Authorization required" popup - click Review Permissions
  6. Select your Google account
  7. Click Advanced, then Go to Newsletter Script (unsafe)
  8. Click Allow

Why "unsafe"?

Google shows this warning because the script isn't verified. Since YOU wrote it, you know it's safe. This is normal for personal scripts!

2.5

Test It!

  1. Go back to your Google Sheet
  2. Refresh the page
  3. You should see a new Newsletter menu in the menu bar!
  4. Click Newsletter > Send Test Email
  5. Check your inbox - you should receive a test email!

Success!

If you got the test email, your newsletter system is working! You can now edit the email content in the sendNewsletter function and send to all subscribers.

Part 3: Create a Signup Form

Let people subscribe through your website

3.1

Deploy as Web App

To let your website add subscribers, you need to deploy the script:

  1. In Apps Script, click Deploy > New deployment
  2. Click the gear icon and select Web app
  3. Set Execute as: "Me"
  4. Set Who has access: "Anyone"
  5. Click Deploy
  6. Copy the Web app URL - you'll need this!
Your URL will look like:
https://script.google.com/macros/s/XXXXX.../exec
3.2

Create the HTML Signup Form

Create a new HTML file for your signup form. Replace YOUR_SCRIPT_URL with your deployed URL:

<body>
    <div class="container">
        <h1>Join My Newsletter</h1>
        <p class="subtitle">Get updates delivered to your inbox!</p>

        <form id="subscribeForm">
            <div class="form-group">
                <label for="name">Your Name</label>

    <script>
        // IMPORTANT: Replace this with YOUR Apps Script URL!
        const SCRIPT_URL = 'YOUR_SCRIPT_URL_HERE';

        document.getElementById('subscribeForm').addEventListener('submit', async function(e) {
            e.preventDefault();

            const submitBtn = document.getElementById('submitBtn');
            const messageDiv = document.getElementById('message');

            // Get form values
            const name = document.getElementById('name').value;
            const email = document.getElementById('email').value;

            // Show loading state
            submitBtn.disabled = true;
            submitBtn.textContent = 'Subscribing...';

            try {
                const response = await fetch(SCRIPT_URL, {
                    method: 'POST',
                    mode: 'no-cors',
                    headers: { 'Content-Type': 'application/json' },
                    body: JSON.stringify({ name, email })
                });

                // Show success (no-cors doesn't return response body)
                messageDiv.className = 'message success';
                messageDiv.textContent = 'Thanks for subscribing! Check your inbox.';
                this.reset();

            } catch (error) {
                messageDiv.className = 'message error';
                messageDiv.textContent = 'Something went wrong. Please try again.';
            }

            // Reset button
            submitBtn.disabled = false;
            submitBtn.textContent = 'Subscribe';
        });
    </script>
</body>
</html>

Part 4: Send Your First Newsletter!

You're ready to send

4.1

Edit the Email Content

In Apps Script, find the sendNewsletter function and edit these parts:

// Change the subject line
const subject = "Your Newsletter Title Here";

// Edit the HTML content
const htmlBody = `
  <h1>Your Headline</h1>
  <p>Hi {{NAME}},</p>
  <p>Write your content here...</p>
`;
4.2

Send It!

  1. Go to your Google Sheet
  2. Click Newsletter > Send Test Email first
  3. Check the test email looks good
  4. When ready, click Newsletter > Send Newsletter to All
  5. A popup will show how many emails were sent!

Remember the Limits!

Free Gmail: 100 emails per day
Google Workspace: 1,500 emails per day
Don't send more than your limit or you'll be blocked temporarily!
