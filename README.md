# Build Your Own Gmail Newsletter

A beginner-friendly tutorial for setting up a free email newsletter using Google Sheets + Google Apps Script. No platforms, no fees, no lock-in.

**Live Site:** [https://vrnico.github.io/newsletter-setup](https://vrnico.github.io/newsletter-setup)

## What This Is

A complete tutorial site that teaches you how to:
- Store subscribers in a Google Sheet
- Send personalized emails via Gmail (100/day free)
- Create a signup form for your website
- Own your entire newsletter infrastructure

## Why?

Newsletter platforms (Substack, Mailchimp, Beehiiv, etc.) are convenient, but they come with trade-offs:
- Your subscriber list lives on their servers
- They can change pricing, algorithms, or policies anytime
- You're building audience on rented land

This tutorial teaches the plumbing so you have options.

## Quick Start

1. Clone or download this repo
2. Open `index.html` in a browser
3. Follow the tutorial

Or just visit the [live site](https://vrnico.github.io/newsletter-setup).

## The Stack

- **Google Sheets** - Your subscriber database
- **Google Apps Script** - Sends emails via Gmail API
- **Vanilla HTML/CSS/JS** - No frameworks, no build step

## Limits

- Free Gmail: 100 emails/day
- Google Workspace: 1,500 emails/day
- No built-in analytics (but you could add tracking)
- You write HTML emails (no drag-and-drop editor)
